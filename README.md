# odin-landing-page
The Odin Project: Landing Page 

In this project I will be practicing HTML, CSS, and specifically getting comfortable with CSS Flexbox layout.

## About project
This a very simple Webpage themed around a platform about finding a team or group to **learn a skill together** and make projects as a **team**.
platform is made for self learners who are wanting to build a programming skill as well as strengthen their team work and communication with their fellow team members.

the page will contain an eye catching introduction to the platform.

## Page details
as it is a very simple beginner lever *practice project* focuses only on the page and how it was made and shaped, i will talk about very simple details about the page structure, and will not talk very professionally about what features does the platform have.

## Page has a
- header with navigation links, logo, and small introduction.
- two more sections with more details about the platform, and some inspiring messages or quotes.
- a section redirecting to the registration page.
- and a footer containing copyright information.
